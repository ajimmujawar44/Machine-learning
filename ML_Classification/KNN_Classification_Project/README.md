# K-Nearest Neighbors (KNN) Classification

## Project Overview
This project demonstrates K-Nearest Neighbors (KNN) Classification using the Social Network Ads dataset.

## Dataset
Social_Network_Ads.csv

## Features
- Data Preprocessing
- Train-Test Split
- Feature Scaling
- KNN Classification
- Confusion Matrix
- Accuracy Score
- Classification Report
- ROC Curve
- Decision Boundary Visualization

## Technologies
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## How to Run

pip install -r requirements.txt

python KNN_Classifier.py
